### What I actually do

Most "full-stack" profiles list five frameworks and no direction. Mine doesn't.

  internal tools that companies run their day on.
- **Applied AI integration** — using LLM APIs to remove manual steps from those workflows
  (auto-categorization, smart replies, document parsing, decision support) — not building

If you're hiring for "developer who understands both the backend of a business and how to
put AI to work inside it," that's the work I want.
